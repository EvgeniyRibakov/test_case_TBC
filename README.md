Решение тестового задания оформлено отдельными скриптами по каждой задаче.

## Состав

- `schema_and_data.sql` - создание таблиц и заполнение тестовыми данными
- `task1.sql` - агрегация заказов по дням и месяцам
- `task2.sql` - статистика по городам и топ-клиент по сумме
- `task3.sql` - помесячная сводка по клиентам
- `task4.sql` - суммы по городам/месяцам + итоги
- `task5.sql` - процедура `sp_AddClient`
- `task6.sql` - генерация диапазона дат и вставка в таблицу

## Порядок запуска

1. Выполнить `schema_and_data.sql`
2. Затем по очереди запускать `task1.sql` ... `task6.sql`

## Примечания

- Все скрипты написаны под SQL Server (T-SQL)
- В каждом файле есть короткие комментарии по логике решения
